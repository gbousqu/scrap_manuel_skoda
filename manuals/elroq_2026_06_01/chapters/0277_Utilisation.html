Utilisation

Envoyer un message texte

  1. Si deux téléphones sont connectés, toucher et sélectionner le téléphone à partir duquel le message SMS sera envoyé.

  2. Passer au contact souhaité dans la liste des contacts disponibles ou favoris.

    ou :

    Toucher et sélectionner le contact souhaité.

  3. Si le contact ne contient qu’un seul numéro d’appel, toucher .

    ou :

    Si le contact contient plusieurs numéros, toucher la position du contact.

  4. Toucher du numéro auquel le message SMS sera envoyé.

  5. Sélectionner le SMS souhaité.

Afficher les messages texte reçus

Le système d’infodivertissement permet uniquement d’afficher les messages SMS reçus pendant la connexion avec le téléphone.

  1. Toucher la barre d’état qui est la plus éloignée du conducteur.

    Les notifications indiquent les messages SMS reçus pendant la connexion au téléphone.

Lorsque la notification SMS est supprimé, il n’est plus possible d’afficher ce message SMS dans le système d’infodivertissement. Le message SMS reste sur le téléphone.

Faire lire le message par l’assistant vocal

  1. Toucher Lecture vocale dans la notification.

Si vous recevez plusieurs messages SMS provenant d’un contact, tous les messages sont affichés dans une notification. Une fois que l’utilisateur a touché Lecture vocale, l’assistant vocal ne lit que le dernier message. Une fois le message lu, la notification est effacée.
