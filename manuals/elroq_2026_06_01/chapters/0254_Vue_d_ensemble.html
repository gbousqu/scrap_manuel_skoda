Afficher l’application Médias

  1. Appuyer sur .

  2. Sélectionner la source médias souhaitée.

ou :

  1. Cliquer sur .

  2. Dans le menu de l’application, toucher la vignette de la source médias directe souhaitée.

Les sources médias directes incluent, par exemple :

  • Clé USB

  • Téléphone Bluetooth connecté

Toucher la source médias directe dans le menu d’application pour ouvrir l’application Médias.

Toucher la source média indirecte dans le menu d’application pour ouvrir l’application de cette source de médias.

En outre, une source médias indirecte peut également être sélectionnée comme source médias dans l’application Médias.

Les sources médias indirectes incluent par exemple :

  • Android Auto et Apple CarPlay

  • La plupart des applications de fournisseur tiers

Sélection d’une source médias

  1. Toucher le symbole de la source médias actuelle dans la barre supérieure et sélectionner la source médias souhaitée.

    Si la radio est sélectionnée comme source, l’application Radio s’ouvre.
