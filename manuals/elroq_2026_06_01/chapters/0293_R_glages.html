Réglages

Connexion via un port USB

Pour établir une connexion, la condition suivante doit être remplie :

  • La version minimale de Android est le 6.

  1. Activez l’Infodivertissement.

  2. Activer l'appareil mobile.

  3. Connecter l'appareil mobile au port USB avant à l'aide d'un câble.

  4. Toucher OK pour ouvrir l’application Android Auto.

    ou :

    Toucher Annuler pour ouvrir l’application Android Auto ultérieurement.

Connexion d’un appareil mobile non couplé via Bluetooth/Wi-Fi depuis le système d’infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth et la visibilité Bluetooth.

  2. Dans le système d’infodivertissement, toucher Gestionnaire de connexion Ajouter un nouvel appareil.

  3. Sélectionner l’appareil souhaité dans la liste des appareils disponibles.

  4. Entrer et confirmer le code PIN le cas échéant.

  5. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  6. Toucher OK pour ouvrir l’application Android Auto.

    ou :

    Toucher Annuler pour ouvrir l’application Android Auto ultérieurement.

Connexion d’un appareil mobile couplé via Bluetooth®/Wi-Fi depuis le système d’infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Dans le système d’infodivertissement, toucher Gestionnaire de connexion.

  3. Dans la liste des appareils connus, sélectionner l'appareil portant le symbole .

  4. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  5. Toucher OK pour ouvrir l’application Android Auto.

    ou :

    Toucher Annuler pour ouvrir l’application Android Auto ultérieurement.

Connexion via Bluetooth®/Wi-Fi depuis l’appareil mobile

Pour établir une connexion, les conditions suivantes doivent être remplies :

  • Dans l'infodivertissement, la fonction Wi-Fi et Bluetooth® est activée.

  • La version minimale de Android est le 11.

  1. Rechercher les appareils Bluetooth® disponibles dans le téléphone.

  2. Sélectionner l’unité Bluetooth® du système d’infodivertissement.

  3. Confirmer le PIN le cas échéant.

  4. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  5. Toucher OK pour ouvrir l’application Android Auto.

    ou :

    Toucher Annuler pour ouvrir l’application Android Auto ultérieurement.

Fermeture de l’application Android Auto

  1. Passer à la liste des applications dans l’application Android Auto et toucher le symbole Škoda.

Coupure de connexion

  1. Dans le système d’infodivertissement, toucher Gestionnaire de connexion.

  2. Toucher une position de téléphone avec une connexion Android Auto active .
