Restriction

Le système d’infodivertissement ne permet pas d’afficher les SMS mémorisés sur le téléphone. Il permet uniquement d’afficher les messages reçus pendant la connexion avec le téléphone.

Le système d’infodivertissement ne permet pas d’écrire des messages SMS. Il est uniquement possible d’envoyer des modèles de messages prédéfinis.

Pour les téléphones fonctionnant avec le système d’exploitation iOS, l’utilisateur doit activer pour la fonction SMS l’option « Afficher les notifications de messages » dans les réglages Bluetooth pour le système d’infodivertissement dans le téléphone.
