Vue d’ensemble

Afficher l'application téléphonique

  1. Appuyer sur .

Si aucune fonction du téléphone n’est disponible, vérifiez que l’appareil mobile est connecté.

Afficher les contacts

  1. Toucher Contacts.

  2. Si deux téléphones sont connectés, toucher et sélectionner le téléphone de votre choix.

Si aucun contact téléphonique n’est affiché, vérifier la demande de confirmation de l’importation des contacts sur le périphérique mobile connecté.

Gérer les téléphones

  1. Toucher Gestionnaire de connexion.

Le téléphone permet l’utilisation des fonctions téléphone

Le téléphone permet d’utiliser le système d’infodivertissement comme lecteur de médias

Le téléphone permet une connexion SmartLink via Android Auto

Le téléphone permet une connexion SmartLink via Apple CarPlay

Affichage d’autres options, telles que :

  • Définir comme téléphone favori

    Le téléphone sera toujours connecté en tant que téléphone principal.

  • Numéro de la personne à contacter

    Lors d’un appel d’urgence, ce numéro est transmis au personnel du centre de secours pour qu’il puisse joindre la personne à contacter.

  • Déconnecter le téléphone

  • Désappairer le téléphone

Pour désappairer le téléphone, balayer vers la gauche avec le doigt en partant du bord droit de la position du téléphone.

Si le symbole de connexion est vert, la connexion est active.
