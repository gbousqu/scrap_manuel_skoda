Utilisation

Connecter l’utilisateur

Après la mise du contact et l’activation de l’infodivertissement, l’écran de connexion initial du dernier compte utilisateur utilisé s’affiche.

  1. Confirmer la connexion.

    En cas de connexion à un compte utilisateur protégé, saisir ensuite le code PIN du véhicule ou le code S-PIN et confirmer la connexion.

Se connecter au compte utilisateur protégé

Certains comptes d’utilisateurs peuvent être protégés. Les comptes protégés sont indiqués dans la liste des utilisateurs par l’icône .

Pour la connexion au compte utilisateur protégé, il est nécessaire d’utiliser le code S-PIN créé lors de la création du compte utilisateur dans l’application MyŠkoda ou le code PIN du véhicule.

Pour régler la protection du compte dans le système d’infodivertissement, toucher Mon profil Modifier la protection par code PIN

Restaurer le code S-PIN

La restauration de votre propre code S-PIN est possible dans l’application MyŠkoda.

Une restauration du code PIN du véhicule n’est pas possible.

Si le profil est protégé par le code PIN du véhicule, il doit être supprimé et ajouté à nouveau en cas de perte du code PIN.
