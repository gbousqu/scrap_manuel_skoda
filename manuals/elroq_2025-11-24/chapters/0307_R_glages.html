Connexion via un port USB

Pour établir une connexion, la condition suivante doit être remplie :

  • La version minimale d'Android est le 6.

  1. Activez l’Infodivertissement.

  2. Activer l'appareil mobile.

  3. Connecter l'appareil mobile au port USB avant à l'aide d'un câble.

  4. Dans l’application SmartLink, sélectionner la connexion via Android Auto.

Connexion d'un appareil mobile non couplé via le Wi-Fi à partir de l'infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Dans l'infodivertissement, appuyer sur appareils mobiles.

  3. Sélectionner l’appareil souhaité dans la liste des appareils disponibles.

  4. Sélectionner le « profil Mode mains libres » Bluetooth®.

  5. Entrer et confirmer le code PIN le cas échéant.

  6. Si 2 autres appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  7. Dans la liste des appareils connus, sélectionner l'appareil portant le symbole .

Connexion d'un appareil mobile couplé via le Wi-Fi à partir de l'infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Dans l’application SmartLink, choisir l’appareil avec le symbole .

    ou :

    Cliquer sur Appareils mobiles.

    Dans la liste des appareils connus, sélectionner l'appareil portant le symbole .

  3. Si 2 autres appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

Connexion via le Wi-Fi depuis l'appareil mobile

Pour établir une connexion, les conditions suivantes doivent être remplies :

  • Dans l'infodivertissement, la fonction Wi-Fi et Bluetooth® est activée.

  • La version minimale d'Android est le 11.

  1. Rechercher les appareils Bluetooth® disponibles dans le téléphone.

  2. Sélectionner l’unité Bluetooth® du système d’infodivertissement.

  3. Confirmer le PIN le cas échéant.

  4. Sur l’écran d’infodivertissement, sélectionner le type de connexion Android Auto.
