Réglages

Activer/désactiver les suggestions de destinations prédictives

Si la suggestion de destinations prédictives est activée, le système peut proposer ces destinations dans les destinations suggérées ainsi que pendant le trajet.

  • Dans l'application Navigation, cliquer sur Fonctions de base Apprendre comport. d'utilisation.

Supprimer les informations pour proposer des destinations prédictives

  • Dans l'application Navigation, cliquer sur Fonctions de base Supprimer le comportement d'utilisation.
