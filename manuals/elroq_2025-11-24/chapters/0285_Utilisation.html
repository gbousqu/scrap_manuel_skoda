Utilisation

Envoyer un message texte

  1. Cliquer sur l'application Téléphone .

    Selon le téléphone connecté, il peut être nécessaire de cliquer sur dans la partie supérieure gauche de l'écran et de sélectionner l'élément de menu SMS.

  2. En bas de l'écran, cliquer sur le champ de saisie du nouveau message.

  3. Rédiger et confirmer un communiqué de texte.

    Le message peut être édité en cliquant sur le champ de texte.

  4. Rechercher ou sélectionner le destinataire du message.

    Plusieurs destinataires peuvent être ajoutés.

  5. Appuyer sur Envoyer.

Afficher les messages texte reçus

  • Cliquer sur l'application Téléphone .

    Selon le téléphone connecté, il peut être nécessaire de cliquer sur dans la partie supérieure gauche de l'écran et de sélectionner l'élément de menu SMS.

Les messages acceptés seront affichés sous forme de messages individuels ou sous forme de conversation en fonction du réglage dans l'application Téléphone dans le menu Afficher les SMS en tant que conversation.

Si les messages acceptés sont affichés en tant que messages individuels :

  1. Pour afficher le menu Messages, balayez le message de droite à gauche.

  2. Dans le menu des messages, cliquer sur .

Si les messages acceptés apparaissent sous forme de conversations :

  • Appuyer sur la conversation pour l'afficher.

  • Maintenir le message souhaité dans la conversation pour afficher le message individuel.

Les fonctions suivantes peuvent être sélectionnées dans le message ou la conversation affiché(e) :

Transmettre le message

Options du message ou de la conversation

Liste des réponses rapides

Le fait de taper sur une réponse rapide l'envoie immédiatement.

Pour obtenir une réponse, toucher le champ de saisie en bas de l’écran.

Dans la conversation affichée, les fonctions se rapportent au dernier message de la conversation.

Faire lire le message par l’assistant vocal

  • Cliquer sur l'application Téléphone .

    Selon le téléphone connecté, il peut être nécessaire de cliquer sur dans la partie supérieure gauche de l'écran et de sélectionner l'élément de menu SMS.

Les messages acceptés seront affichés sous forme de messages individuels ou sous forme de conversation en fonction du réglage dans l'application Téléphone dans le menu Afficher les SMS en tant que conversation.

Si les messages acceptés sont affichés en tant que messages individuels :

  • Toucher le message souhaité.

Si les messages acceptés apparaissent sous forme de conversations :

  1. Toucher une conversation spécifique.

  2. Toucher le message souhaité dans la conversation.
