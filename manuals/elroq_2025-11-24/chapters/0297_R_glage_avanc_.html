Réglage avancé

Changer le nom du point d’accès de l’infodivertissement

  1. Appuyer sur Wi-Fi Système d'infodivertissement comme point d’accès Nom du réseau :.

  2. Entrer le nom et confirmer.

Connexion rapide à l'aide du code QR

  1. Appuyer sur Wi-Fi Connexion rapide avec le système d’infodivertissement.

  2. Sur l’appareil mobile, activer l’option de connexion via code QR.

  3. Lire le code QR affiché sur l’écran de l’infodivertissement avec l’appareil mobile.

  4. Attendre que la connexion soit établie, confirmer les éventuels messages à ce sujet.
