Réglages

Types de connexion entre le téléphone et l'Infodivertissement

La connexion du téléphone est établie de la manière suivante en fonction du nombre de téléphones actuellement connectés à l’Infodivertissement :

  • Téléphone principal – lorsqu’aucun téléphone n’est connecté.

  • Téléphone supplémentaire – lorsqu'un téléphone est connecté

  • Choix du téléphone à remplacer - lorsque 2 téléphones sont connectés.

Commutation entre le téléphone principal et le téléphone supplémentaire

  1. Appuyer sur Téléphone dans n’importe quel menu de l’application.

  2. Sélectionner le téléphone souhaité.

Connecter la tablette à l’Infodivertissement

  1. Rechercher les appareils Bluetooth® disponibles dans le téléphone.

  2. Sélectionner l’unité Bluetooth® du système d’infodivertissement.

    Le nom de l'unité peut être communiqué en cliquant sur Appareils mobiles Nom de l'appareil :.

  3. Entrer et confirmer le code PIN le cas échéant.

Connecter l’Infodivertissement à un téléphone non couplé

  1. Assurer la visibilité Bluetooth® de l’appareil mobile.

  2. Cliquer sur Appareils mobiles.

  3. Sélectionner l’appareil souhaité dans la liste des appareils disponibles.

  4. Sélectionner le profil Bluetooth® souhaité.

  5. Entrer et confirmer le code PIN le cas échéant.

  6. Si 2 appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

Connecter l’Infodivertissement à un téléphone déjà couplé

La connexion s’effectue automatiquement à la mise du contact. Le téléphone doit être connecté manuellement comme suit :

  1. Cliquer sur Appareils mobiles.

  2. Sélectionner l’appareil souhaité dans la liste des appareils connus.

  3. Sélectionner le profil Bluetooth® souhaité.

  4. Si 2 appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.
