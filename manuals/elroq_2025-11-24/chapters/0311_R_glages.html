Réglages

Connexion via un port USB

  1. Activez l’Infodivertissement.

  2. Activer l'appareil mobile.

  3. Connecter l'appareil mobile au port USB avant à l'aide d'un câble.

  4. Sélectionner la connexion au moyen de Apple CarPlay.

Connexion via le Wi-Fi depuis l'infodivertissement

  1. Sur l'appareil mobile Apple, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Cliquer sur Appareils mobiles.

  3. Dans la liste affichée, rechercher et sélectionner l’appareil souhaité portant le symbole .

  4. Suivre les instructions apparaissant sur l’écran du système d’infodivertissement et confirmer le couplage Bluetooth®.

  5. Si un autre appareil est connecté au système d’infodivertissement, sélectionner l’appareil à remplacer.

Connexion via le Wi-Fi depuis l'appareil mobile

Pour établir une connexion, les conditions suivantes doivent être remplies :

  • La version minimale d'iOS est le 9.

  • Il n'y a pas d'autre appareil mobile à connecter via SmartLink.

  • Dans l'infodivertissement, la fonction Wi-Fi et Bluetooth® est activée.

  1. Dans l'appareil mobile, activer la fonction CarPlay.

  2. Maintenir la touche enfoncée sur le volant multifonction.

    L'écran d’infodivertissement affiche un message concernant la connexion au nom de l'unité Bluetooth® de l'infodivertissement.

  3. Rechercher les appareils Bluetooth® disponibles dans le téléphone.

  4. Sélectionner l’unité Bluetooth® du système d’infodivertissement.

  5. Confirmer le PIN le cas échéant.

  6. Sur l’écran d’infodivertissement, sélectionner le type de connexion Apple CarPlay.
