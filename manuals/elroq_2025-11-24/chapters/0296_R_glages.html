Connecter un appareil mobile au point d'accès de l’infodivertissement

  1. Activer le point d’accès Wi-Fi (WLAN) de l’Infodivertissement dans le menu WLAN Système d'infodivertissement comme hotspot Utiliser en tant que Hotspot.

  2. Activer le Wi-Fi sur l’appareil mobile.

  3. Dans la liste des appareils Wi-Fi disponibles, trouver le Wi-Fi de l'infodivertissement.

    Le nom de l’infodivertissement est indiqué dans le menu Wi-Fi Système d'infodivertissement comme hotspot Nom du réseau :.

  4. Sélectionnez le hotspot de l’Infodivertissement et entrez le mot de passe.

    Le mot de passe: est indiqué dans le menu Wi-Fi Système d'infodivertissement comme hotspot Mot de passe:.

  5. Confirmer l’établissement de la connexion.

Connecter l'infodivertissement au point d'accès Wi-Fi de l'appareil mobile

  1. Activer l’unité Wi-Fi de l’infodivertissement dans le menu Wi-Fi Wi-Fi : Activer Rechercher Wi-Fi.

  2. Sélectionner le hotspot souhaité dans la liste des hotspots disponibles.

  3. Entrer le mot de passe pour vous connecter à l’appareil mobile.

L’infodivertissement enregistre les mots de passe des points d’accès auxquels il s’est précédemment connecté.

Changer le mot de passe pour la connexion au point d’accès de l’infodivertissement

Le mot de passe doit être composé de 12 à 63 caractères. Il doit contenir au moins une lettre minuscule, une lettre majuscule et un chiffre.

  1. Appuyer sur Wi-Fi Système d'infodivertissement comme point d’accès Mot de passe :.

  2. Entrer le mot de passe et confirmer.

Transmission de données avec l'appareil connecté

  • Pour activer/désactiver, appuyer brièvement sur la transmission de données , appuyer sur Wi-Fi Système d'infodivertissement comme hotspot Appareils connectés .
