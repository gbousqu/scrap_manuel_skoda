Utilisation

Le clavier est affiché automatiquement lorsque l'infodivertissement requiert une entrée de caractères.

Guide de recherche

La recherche des entrées correspondantes est effectuée pendant la saisie des caractères.

L’entrée à rechercher peut être entrée avec des signes diacritiques.

Afficher les langues clavier supplémentaires

Pour utiliser les langues de clavier supplémentaires, les langues souhaitées doivent d’abord être ajoutées dans le menu Langues de clavier supplémentaires.

  • Pour les utiliser, appuyer sur sur le clavier à plusieurs reprises jusqu’à ce que le clavier de la langue souhaitée s’affiche.

Autres variantes de caractères

  1. Maintenir le caractère enfoncé pour afficher plus de variantes de caractères.

  2. Sélectionner le caractère souhaité.
