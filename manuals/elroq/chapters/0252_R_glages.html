Réglages

Activer/désactiver la réception des informations routières (TP) des stations

  • Dans l'application Radio/Médias, cliquer sur Radio Radioguidage (TP).

Activer/désactiver les infos routières DAB supplémentaires de la station DAB

Si la fonction est activée, les autres annonces sont reçues en tant qu’informations trafic. Il s'agit par ex. d'informations météorologiques, de reportages sportifs, de nouvelles financières, etc.

  • Dans l'application Radio/Médias, cliquer sur Radio Infos routières DAB supplémentaires.

Activer/désactiver la réception de texte radio

Lorsque la fonction est activée, vous pouvez voir plus d’informations sur le contenu diffusé à partir de la station que vous écoutez.

  • Dans l'application Radio/Médias, cliquer sur Radio Radiotexte.

Touches /

Le type pour le changement de station à l’aide des touches de fonction / est réglée comme suit :

  1. Dans l'application Radio/Médias, cliquer sur Radio Changer de station :.

  2. Sélectionner l’option permettant de basculer entre les stations mémorisées sur les touches de stations ou entre toutes les stations.
