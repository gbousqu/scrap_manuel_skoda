Usage prévu

Les activités suivantes sont possibles dans le menu Shop de l’infodivertissement.

  • Commander de nouvelles fonctions et services

  • Prolonger les fonctions et services achetés

Paquets de données

Les appareils connectés via le Wi-Fi peuvent également accéder aux packs de données.

Il est possible de trouver de plus amples informations sur les services Škoda Connect concernant la fonction de la connexion de données, l'enregistrement de l'utilisateur pour l'utilisation des packs de données dans le véhicule, la disponibilité de la connexion de données et un aperçu des packs de données et des pays avec la possibilité de les obtenir sur le site Web suivant :

https://go.skoda.eu/cubictelecom

Les fonctions et services peuvent être soumis à une limitation de la durée de fonctionnement ou peuvent être illimités.

Les fonctions et services peuvent également être achetés rétroactivement sur le site Web Škoda Connect Shop.

https://go.skoda.eu/connect-shop
