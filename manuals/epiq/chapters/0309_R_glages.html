Activation des services Škoda Connect

Les services sont activés en enregistrant l'utilisateur principal.

Certains services Škoda Connect nécessitent des données mobiles supplémentaires pour pouvoir être utilisés.

La licence Škoda Connect Premium inclut un forfait de données supplémentaire fourni par l’opérateur de téléphonie mobile.

Activation du forfait de données

  1. Dans le système d’infodivertissement, toucher Forfaits de données.

  2. Scanner le code QR avec le téléphone mobile et suivre les instructions qui s’affichent.
