Activation automatique

Lane Assist est toujours automatiquement activé à la mise du contact (uniquement valable pour certains pays) .

Activation/désactivation du système

  1. Dans le système d’infodivertissement, toucher Voies .

Le système peut également être activé et désactivé dans le menu. Pour cela, appuyer sur la touche dans la console centrale.

L’activation ou la désactivation du système est possible dans le centre de commande Centre de commande du système d’infodivertissement si le bouton de fonction Lane Assist est configuré.

Activation/désactivation du mode réduit

  1. Dans le système d’infodivertissement, toucher Voies Mode réduit.

La fonction peut également être activée et désactivée dans le menu. Pour cela, appuyer sur la touche dans la console centrale.

Après la coupure du contact, le dernier réglage du mode réduit est automatiquement enregistré.
