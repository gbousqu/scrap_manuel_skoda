Trouver un parc de stationnement

  1. Toucher .

  2. Pour une recherche détaillée, toucher Filtre de recherche et sélectionner les critères requis.

Si des données en ligne sont disponibles, il est possible d’afficher des informations détaillées sur certains parcs de stationnement, par exemple sur la disponibilité des places de stationnement.

Le nombre de places de stationnement disponibles sur les parkings ou dans les rues est représenté par le code couleur suivant :

  • Places de stationnement majoritairement libres

  • Environ la moitié des places de stationnement libres

  • Places de stationnement majoritairement occupées

  • Aucune information en temps réel disponible sur les places de stationnement dans les rues

Rechercher un lieu de recharge

  1. Toucher .

  2. Pour une recherche détaillée, toucher Filtre de recherche et sélectionner les critères requis.

Si des données en ligne sont disponibles, il est possible d’afficher des informations détaillées sur certaines stations de recharge, par exemple sur la disponibilité des lieux de recharge.

L’occupation des lieux de recharge est indiquée par le code couleur suivant :

  • Places de stationnement majoritairement libres

  • Environ la moitié des places de stationnement libres

  • Places de stationnement majoritairement occupées
