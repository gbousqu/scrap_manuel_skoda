Réglages

Connecter un appareil mobile à l'infodivertissement

  1. Dans le système d’infodivertissement, toucher et s’assurer que l’option de menu Bluetooth est activée.

  2. Rechercher les périphériques Bluetooth disponibles sur l'appareil mobile.

  3. Sélectionner l’unité Bluetooth® du système d’infodivertissement.

    • Pour afficher ou modifier le nom de l’unité, toucher Bluetooth Nom du véhicule dans le système d’infodivertissement.

  4. Entrer et confirmer le code PIN le cas échéant.

Connecter l'infodivertissement à un appareil mobile non apparié

  1. Assurer la visibilité Bluetooth® de l’appareil mobile.

  2. Dans le système d’infodivertissement, toucher Gestionnaire de connexion Ajouter un nouvel appareil.

  3. Sélectionner l’appareil souhaité dans la liste des appareils disponibles.

  4. Sélectionner le type de connexion souhaité.

  5. Entrer et confirmer le code PIN le cas échéant.

  6. Si deux appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

Connecter l'infodivertissement à un appareil mobile précédemment appairé

La connexion avec un appareil mobile déjà couplé se fait automatiquement après la mise du contact. Si nécessaire, l'appareil peut être connecté manuellement comme suit :

  1. Dans le système d’infodivertissement, toucher Gestionnaire de connexion.

  2. Sélectionner l’appareil souhaité dans la liste des appareils connus.

  3. Sélectionner le type de connexion souhaité.

  4. Si deux appareils sont connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.
