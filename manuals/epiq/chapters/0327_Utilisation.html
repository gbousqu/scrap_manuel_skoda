Utilisation

L’application Connect Store permet de gérer les services et applications Škoda Auto ainsi que les applications de fournisseurs tiers.

Services et applications Škoda Connect

  1. Dans l’application Connect Store, toucher l’onglet Škoda Connect.

    Une liste des services et applications Škoda Auto disponibles et actifs s’affiche.

Toucher un service ou une application pour afficher les détails et d’autres options associées.

Le téléchargement de services et d’applications Škoda Auto n’utilise aucune donnée du forfait de données de l’utilisateur.

Applications externes

Les applications externes sont des applications développées par des partenaires Škoda Auto.

Installer une application externe :

  1. Dans l’application Connect Store, toucher l’onglet Applis.

  2. Toucher Installer sur la position de l’application souhaitée.

  3. Confirmer le téléchargement de l’application via une connexion de données disponible.

Supprimer une application externe :

  1. Dans l’application Connect Store, toucher l’onglet Applis.

    ou :

    Dans l’application Connect Store, toucher l’onglet Mes applis.

  2. Toucher la position de l’application à supprimer.

  3. Toucher Désinstaller et confirmer la suppression de l’application.

Afficher les applications externes installées :

  1. Dans l’application Connect Store, toucher l’onglet Mes applis.

    Le menu affiché contient une liste des applications installées et des applications pour lesquelles des mises à jour sont disponibles.

Afficher les applications externes précédemment installées :

  1. Dans l’application Connect Store, toucher Applis précédemment installées.

    Une liste des applications qui ont été installées par le passé, mais qui ne le sont plus actuellement, s’affiche.

Mise à jour des applications

Si une mise à jour des applications est disponible, une notification correspondante s’affiche dans le centre de notifications.

Démarrer la mise à jour :

  1. Dans le centre de notifications du système d’infodivertissement, toucher la notification de la mise à jour disponible.

    ou :

    Dans l’application Connect Store, toucher l’onglet Mes applis.

  2. Pour mettre à jour toutes les applications disponible, toucher Mettre à jour tout.

    ou :

    Pour mettre à jour une application précise, toucher Mises à jour sur la position de l’application.
