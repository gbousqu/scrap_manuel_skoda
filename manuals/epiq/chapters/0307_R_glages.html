Activer/désactiver la connexion de données pour configurer un point d'accès dans le véhicule

La connexion de données prépayée peut être utilisée pour configurer un point d'accès dans le véhicule.

La connexion de données peut être partagée par Wi-Fi avec jusqu’à 8 appareils connectés.

  1. Dans le système d’infodivertissement, toucher .

  2. Activer ou désactiver le curseur dans l’option de menu Point d’accès du véhicule.
