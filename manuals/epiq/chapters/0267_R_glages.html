Connecter un appareil mobile au point d'accès de l’infodivertissement

  1. Pour activer le point d’accès Wi-Fi du système d’infodivertissement, toucher et, dans l’option de menu Point d’accès du véhicule, faire glisser le curseur sur la position « Activé ».

  2. Activer le Wi-Fi sur l’appareil mobile.

  3. Dans la liste des appareils Wi-Fi disponibles, chercher le Wi-Fi du système d’infodivertissement.

    • Pour afficher ou modifier le nom du Wi-Fi, toucher Point d’accès du véhicule Nom du véhicule.

  4. Sélectionnez le hotspot de l’Infodivertissement et entrez le mot de passe.

    • Pour afficher ou modifier le mot de passe, toucher Point d’accès du véhicule Point d’accès du véhicule mot de passe dans le système d’infodivertissement.

  5. Confirmer l’établissement de la connexion.

Connecter l'infodivertissement au point d'accès Wi-Fi de l'appareil mobile

Cette fonction peut ne pas être disponible dans certains pays.

  1. Toucher Wi-Fi.

  2. Activer le Wi-Fi.

  3. Sélectionner le point d’accès souhaité dans la liste des points d’accès disponibles.

  4. Entrer le mot de passe pour vous connecter à l’appareil mobile.

Le système d’infodivertissement enregistre les données de connexion des points d’accès auxquels il s’est précédemment connecté.

Changer le mot de passe pour la connexion au point d’accès de l’infodivertissement

Le mot de passe doit être composé de 12 à 63 caractères. Il doit contenir au moins une lettre minuscule, une lettre majuscule et un chiffre.

  1. Toucher Point d’accès du véhicule Point d’accès du véhicule mot de passe.

  2. Entrer le mot de passe et confirmer.

Activation et désactivation de la transmission des données avec l’appareil connecté

  1. Toucher Point d’accès du véhicule Partager la connexion Internet.
