Mode de fonctionnement

L’enregistrement de l’utilisateur permet d’obtenir le compte utilisateur Škoda ID qui peut être utilisé dans toutes les applications et tous les services Škoda.

L’enregistrement de l’utilisateur aux services Škoda Connect peut se faire dans l’application MyŠkoda.

  1. Pour l’enregistrement, suivre les instructions dans l’application MyŠkoda.

Un code S-PIN est créé lors de l’enregistrement.

Le code S-PIN permet d’autoriser les demandes au sein de certains services Škoda Connect et de se connecter au compte utilisateur protégé.
