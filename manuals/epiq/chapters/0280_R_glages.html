Réglages

Connexion d’un appareil mobile non couplé via Bluetooth/Wi-Fi depuis le système d’infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Dans le système d’infodivertissement, toucher Gestionnaire de connexion Ajouter un nouvel appareil.

  3. Dans la liste affichée, rechercher et sélectionner l’appareil souhaité.

  4. Suivre les instructions s’affichant sur le téléphone et sur l’écran du système d’infodivertissement, puis confirmer le couplage Bluetooth®.

  5. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

Connexion d’un appareil mobile couplé via Bluetooth/Wi-Fi depuis le système d’infodivertissement

  1. Sur l'appareil mobile, activer le Bluetooth® et la visibilité Bluetooth®.

  2. Dans le système d’infodivertissement, toucher Gestionnaire de connexion Ajouter un nouvel appareil.

  3. Dans la liste affichée, rechercher et sélectionner l’appareil souhaité.

  4. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  5. Toucher OK pour ouvrir l’application Apple CarPlay.

    ou :

    Toucher Annuler pour ouvrir l’application Apple CarPlay ultérieurement.

Connexion via Bluetooth/Wi-Fi depuis l’appareil mobile

Pour établir une connexion, les conditions suivantes doivent être remplies :

  • La version minimale de iOS est le 9.

  • Il n'y a pas d'autre appareil mobile à connecter via SmartLink.

  • Dans l'infodivertissement, la fonction Wi-Fi et Bluetooth est activée.

  1. Dans l'appareil mobile, activer la fonction Bluetooth.

  2. Rechercher les appareils Bluetooth disponibles dans le téléphone.

  3. Sélectionner l’unité Bluetooth du système d’infodivertissement.

  4. Confirmer le PIN le cas échéant.

  5. Suivre les instructions s’affichant sur le téléphone et sur l’écran du système d’infodivertissement, puis confirmer le couplage Bluetooth®.

  6. Si deux autres appareils sont déjà connectés au système d’infodivertissement, sélectionner l’appareil à remplacer.

  7. Toucher OK pour ouvrir l’application Apple CarPlay.

    ou :

    Toucher Annuler pour ouvrir l’application Apple CarPlay ultérieurement.

Fermeture de l’application Apple CarPlay

  1. Passer à la liste des applications dans l’application Apple CarPlay et toucher le symbole Škoda.

Coupure de connexion

  1. Dans le système d’infodivertissement, toucher Gestionnaire de connexion.

  2. Toucher une position de téléphone avec une connexion Apple CarPlay active .
