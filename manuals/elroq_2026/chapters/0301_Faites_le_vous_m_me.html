Mise à jour automatique de la base de données de navigation

La base de données de navigation peut être automatiquement mise à jour lorsque le service Infotainment Online est acheté Services Škoda Connect.

Activation et désactivation de la mise à jour automatique

  1. Toucher Base de données et clause de non-responsabilité Mise à jour automatique.

Mise à jour manuelle de la base de données de navigation

Les informations concernant la mise à jour du logiciel sont disponibles auprès de nos partenaires Škoda ou sur les pages Internet Škoda suivantes.

https://go.skoda.eu/updateportal

Sur ce site Internet, il est également possible de signaler des données erronées ou inexactes dans la base de données de navigation.

Définir la version de la base de données de navigation

  1. Toucher Base de données et clause de non-responsabilité.

    La version de la base de données est indiquée dans l’option de menu Base de données.
