Utilisation

Types d’établissement de la connexion de données

  • Par une connexion de l’infodivertissement au point d'accès externe avec accès Internet via Wi-Fi

ou :

  • Au moyen d'un paquet de données pour la carte eSIM dans le véhicule

Forfait de données

Certains services Škoda Connect nécessitent des données mobiles supplémentaires pour pouvoir être utilisés.

La licence Škoda Connect Premium inclut un forfait de données supplémentaire fourni par l’opérateur de téléphonie mobile.

Activation du forfait de données :

  1. Dans le système d’infodivertissement, toucher Forfaits de données.

  2. Scanner le code QR avec le téléphone mobile et suivre les instructions qui s’affichent.

Le système d’infodivertissement connecté à un point d’accès Wi-Fi externe utilise les données de ce point d’accès externe.

Si le système d’infodivertissement n’est pas connecté à un point d’accès Wi-Fi externe, il utilise les données du forfait de données de l’utilisateur.
