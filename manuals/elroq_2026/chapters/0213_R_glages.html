Réglages

Afficher une autre page

  • Faire glisser le doigt sur l’écran vers la gauche ou vers la droite.

Modifier le menu préféré

  1. Maintenir le doigt sur l'une des vignettes du menu HOME.

  2. Sélectionner la vignette à modifier en appuyant brièvement dessus.

  3. Dans la liste des fonctions, sélectionner la fonction à mémoriser dans la vignette sélectionnée.

    Seules les fonctions pouvant être mémorisées dans la vignette sélectionnée sont affichées dans la liste.

  4. Appuyer brièvement sur Terminé.

Ajouter une page du menu HOME

  1. Maintenir le doigt sur l'une des vignettes du menu principal.

  2. Appuyer sur .

  3. Sélectionner la position souhaitée pour la page.

  4. Appuyer brièvement sur OK Terminé.

Le menu HOME peut contenir un maximum de 5 pages.

Retirer une page du menu HOME

  1. Maintenir le doigt sur l'une des vignettes du menu principal.

  2. Appuyer brièvement sur Supprim. page OK Terminé.
