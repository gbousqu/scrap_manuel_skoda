Dégager les composants sous le capot avant

Retirer le rangement

  1. Saisir le rangement dans la zone A et le détacher dans le sens de la flèche.

  2. Sortir le rangement.

Retirer le cache gauche de la zone sous le capot avant

Le cache gauche doit être retiré pour accéder à la batterie 12 V, aux fusibles et au réservoir de liquide de frein.

  1. Ouvrir avec précaution la partie supérieure du bouchon du réservoir de lave-glace.

  2. Retirer le bouchon et le tamis.

  1. Saisir le cache dans la zone A et détacher les languettes de raccordement avec le cache adjacent dans le sens de la flèche.

  2. Retirer le cache.

Retirer le cache droit de la zone sous le capot avant

Pour accéder au vase d’expansion du liquide de refroidissement, retirer le cache droit.

  1. Saisir le cache dans la zone A et le détacher dans le sens de la flèche. Le symbole se trouve à cet endroit sur le cache.

  2. Retirer le cache.

Mettre en place le cache droit dans la zone sous le capot avant

  1. Mettre en place le logement A du cache sous le support B.

  2. Appuyer sur le cache dans la zone C jusqu’à ce qu’il s’enclenche. Le symbole se trouve à cet endroit sur le cache.

Mettre en place le cache gauche dans la zone sous le capot avant

  1. Mettre en place le logement A du cache sous le support B.

  2. Ajuster le cache de sorte que les languettes de raccordement soient alignées avec les languettes du cache voisin. Appuyer sur le cache dans la zone C jusqu’à ce qu’il s’enclenche.

  1. Mettre le bouchon en place sur le raccord du réservoir de lave-glace jusqu’à ce qu’il s’enclenche.

  2. Insérer le tamis dans la tubulure du réservoir.

  3. Fermer le couvercle du récipient.

Mettre en place le rangement

  1. Mettre en place les logements A du rangement dans les supports B.

  2. Appuyer sur le rangement jusqu’à ce que les logements C s’enclenchent sur la barre D.

AVERTISSEMENT

  • Appuyer avec la main, ne pas frapper sur les caches et le rangement.
