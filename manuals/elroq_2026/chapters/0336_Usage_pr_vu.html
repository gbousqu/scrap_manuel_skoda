Usage prévu

La personnalisation permet d’enregistrer les réglages des fonctions personnalisées du véhicule dans le profil utilisateur actif.

Dans les profils avec le compte utilisateur Škoda ID, toute modification des réglages de fonction est automatiquement enregistrée dans le compte utilisateur après la coupure du contact tant que le véhicule se trouve à portée d’un réseau mobile sous contrat.

La connexion au compte utilisateur dans un autre véhicule de la marque Škoda permet de synchroniser les fonctions personnalisées associées au compte utilisateur en fonction du type de véhicule, de l’équipement et de l’infodivertissement.
