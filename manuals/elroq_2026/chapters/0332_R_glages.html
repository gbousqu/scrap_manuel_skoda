Réglages

Activer/désactiver le mode Demo

  • Dans l'application Navigation, cliquer sur Fonctions de base Mode démo.

Définir le point de départ du mode Demo

  1. Activer le mode Demo.

  2. Dans l’application Navigation, appuyer brièvement sur Fonctions de base Définir le point de départ du mode démo.

  3. Rechercher le lieu devant être défini comme point de départ pour le mode Demo.

ou :

  1. Activer le mode Demo.

  2. Sélectionner ou rechercher le lieu devant être défini comme point de départ pour le mode Demo.

    Une fenêtre contenant les détails du lieu sélectionné apparaît.

  3. Agrandir la fenêtre en appuyant brièvement sur la barre située sur le bord gauche de la fenêtre.

  4. Appuyer brièvement sur Démo ou Mode démo.
