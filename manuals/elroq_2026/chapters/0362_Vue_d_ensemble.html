Vue d’ensemble

Mode Eco

Le mode ECO est adapté à un style de conduite économe. La puissance du moteur est limitée, la vitesse maximale est réduite à 130 km/h. Le Climatronic passe en mode économie.

Mode Comfort

Le mode confort est adapté aux déplacements sur les routes avec une surface en mauvaise état et aux longs trajets sur autoroute.

Mode Normal

Le mode Normal est adapté à une conduite normale.

Mode Sport

Le mode Sport est adapté à une conduite sportive.

Mode Traction

Les informations suivantes s’appliquent aux véhicules à transmission intégrale.

Le mode Traction est adapté aux trajets sur des routes lisses. La puissance est répartie de manière optimale entre les essieux avant et arrière.

Mode Individual

Le mode Individual permet de régler de manière personnalisée certains systèmes du véhicule.
