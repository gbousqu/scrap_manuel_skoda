Gestion des utilisateurs

  1. Dans le système d’infodivertissement, toucher Tous les utilisateurs.

Ajouter un profil :

  1. Appuyer sur .

  2. Suivre les instructions à l’écran de l'Infodivertissement.

Changer de profil :

  1. Toucher le profil souhaité.

  2. Si le profil est protégé par un code S-PIN ou un code PIN du véhicule, saisir le code PIN requis.

Supprimer le profil :

  1. Maintenir le doigt sur le profil à supprimer.

  2. Toucher et confirmer la suppression du profil.

Types de profils utilisateurs

Utilisateur principal

  • L’utilisateur principal correspond à un utilisateur enregistré pour lequel les services Škoda Connect sont activés et qui possède un véhicule associé à son compte. Le véhicule ne peut avoir qu’un seul utilisateur principal actif.

  • Tous les réglages sont enregistrés dans le compte en ligne personnel de l’utilisateur principal.

  • L’utilisateur principal peut tirer pleinement parti des fonctionnalités de personnalisation et de leur synchronisation.

  • L’utilisateur principal a le contrôle total sur le réglage et l’utilisation du véhicule.

  • L'utilisateur principal peut accéder aux fonctions à distance du véhicule.

  • L'utilisateur principal peut effectuer la gestion des utilisateurs avec le compte en ligne Škoda ID dans le menu Gestion des utilisateurs et dans l’application MyŠkoda.

  • Il n’est pas possible de modifier ou de supprimer le profil d’utilisateur principal. La suppression n’est possible qu’avec la réinitialisation du système d’infodivertissement aux réglages d’usine.

ID Škoda de l’utilisateur

  • L’utilisateur n’est pas l’utilisateur principal mais possède son propre compte en ligne Škoda ID.

  • L'utilisateur peut utiliser pleinement la fonction de personnalisation ainsi que certains services dans l'application MyŠkoda.

  • Tous les réglages sont enregistrés dans le compte en ligne personnel de l’utilisateur inscrit.

Utilisateur local

  • Le profil utilisateur local n’est pas connecté au compte en ligne Škoda ID et n’est pas synchronisé avec les services Škoda Connect.

  • Toutes les modifications des réglages liées à ce profil restent enregistrées uniquement dans le véhicule.

  • Il est possible d’ajouter le compte Škoda ID au profil utilisateur local. Le profil utilisateur local est alors remplacé par le profil utilisateur avec le compte Škoda ID.

Invité

  • Le profil invité n’est jamais synchronisé avec les services Škoda Connect.

  • Toutes les modifications des réglages liées à ce profil restent enregistrées uniquement dans le véhicule.

Utilisateur temporaire

  • Le profil utilisateur temporaire est totalement anonyme et aucun réglage ou personnalisation n’est enregistré(e).

  • Si le profil utilisateur temporaire est actif lorsque le véhicule est quitté, le profil utilisateur Invité est automatiquement sélectionné lors du prochain trajet.

Gérer l’utilisateur actuel

  1. Dans le système d’infodivertissement, toucher Mon profil.

Les options suivantes par exemple peuvent être définies dans le menu affiché :

  • Paramètres des comptes liés – permet de gérer les liens de profil avec les comptes tiers.

  • Réglages de protection du profil – permet de définir la manière dont le profil est protégé ou d’annuler la protection.

  • Utilisation d’un utilisateur temporaire – permet de passer à un profil utilisateur totalement anonyme qui n’enregistre pas les réglages.

  • Ajouter un compte Škoda ID – l’ajout d’un compte Škoda ID modifie le profil utilisateur local en un profil utilisateur avec un compte Škoda ID.

  • Synchronisation d’un compte en ligne – la synchronisation permet d’enregistrer et de transférer les modifications apportées aux fonctions personnalisées d’un compte utilisateur dans un autre véhicule.

  • Personnalisation – l’option de menu n’est disponible que pour le profil invité. Crée un nouveau profil avec la personnalisation provenant du profil invité.
