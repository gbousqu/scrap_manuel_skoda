Activation automatique

Valable pour certains pays seulement.

L'activation de l'alerte de sortie se fait automatiquement à la mise du contact.

Pour afficher le menu d’activation ou de désactivation du système, procéder comme suit :

  1. Appuyer sur la touche dans la console centrale.

  2. Appuyer sur dans l’infodivertissement.

  3. Sélectionner le point de menu Avertissement de sortie.

ou :

  1. Engager la marche arrière.

  2. Appuyer sur dans l’infodivertissement.

  3. Sélectionner le point de menu Avertissement de sortie.

ou :

  1. Dans le système d’infodivertissement, toucher .

  2. Sélectionner le point de menu Avertissement de sortie.
