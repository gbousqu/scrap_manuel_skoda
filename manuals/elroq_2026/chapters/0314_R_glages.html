Réglages

Régler les instructions de navigation

  1. Appuyer sur .

  2. Dans les options de menu Instructions de navigation, Volume instructions et Instructions de navigation pendant appel, sélectionner le réglage souhaité.

Régler les options d'itinéraire

  1. Toucher Options d’itinéraire.

  2. Régler les options de menu souhaitées.

Activer/désactiver l'ajout automatique d'une borne de recharge sur l'itinéraire

Lorsque la fonction est activée, l'infodivertissement ajoute automatiquement une borne de recharge sur l’itinéraire si nécessaire.

  1. Toucher Réglages de recharge Planificateur d’itinéraire pour véhicule électrique Planificateur d’itinéraire pour véhicule électrique.

Définir un filtre pour les stations de recharge préférées sur la carte

  1. Toucher Réglages de recharge Filtres.

  2. Configurez les paramètres souhaités de la borne de recharge.

Activer/désactiver le filtre des stations de recharge préférées sur la carte

Si le filtre est activé, seules les bornes de recharge qui correspondent aux paramètres définis sont mises en surbrillance sur la carte.

  1. Toucher Réglages de recharge Filtres Filtre de recherche.

Paramétrer l’autonomie minimale

  1. Toucher Réglages de recharge Planificateur d’itinéraire pour véhicule électrique.

  2. Dans l’option de menu État de charge minimal à destination, régler la valeur souhaitée.
