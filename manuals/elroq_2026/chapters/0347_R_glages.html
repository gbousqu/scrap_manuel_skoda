Réglages

Les services légalement requis, par exemple l'appel d'urgence, ne peuvent pas être désactivés.

L'utilisateur du véhicule peut activer/désactiver le mode en ligne et l’option Envoyer données de localisation.

Informations sur le paramétrage de la confidentialité

Activer/désactiver Mode en ligne dans le menu Utilisateurs

  1. Dans l'infodivertissement, cliquer sur Paramètres de confidentialité.

  2. Sélectionner l’élément de menu souhaité.

L'icône des options activées/désactivées Mode en ligne et Envoyer données de position s'affiche dans la barre d'état.

Mode en ligne désactivé

  • Tous les services Škoda Connect autres que ceux requis par la loi sont désactivés.

  • Tous les services de localisation sont désactivés. Les informations relatives à la position du véhicule ne sont pas partagées.

  • La transmission d’informations via eSIM est désactivée.

  • Aucune mise à jour en ligne n'est disponible.

Mode en ligne activé et Envoyer données de position activé

  • Tous les services Škoda Connect sont activés.

  • Les informations relatives à la localisation du véhicule sont disponibles pour tous les services pour lesquels il est nécessaire de connaître la localisation du véhicule.

  • Les informations sur la localisation du véhicule sont accessibles dans l’application MyŠkoda.

Mode en ligne activé et Envoyer données de position désactivé

  • Tous les services Škoda Connect, à l'exception des services qui utilisent la localisation du véhicule, sont activés. Les informations relatives à la position du véhicule ne sont pas partagées.

  • La transmission d’informations via eSIM est activée.

  • Toutes les fonctions qui nécessitent une connexion via eSIM sont actives.

Collecte de données anonymisées

Si l’option Envoyer données de position est activée, le véhicule peut collecter et envoyer des données anonymisées afin d'améliorer les services de tiers sélectionnés.

Il s’agit de données tout à fait anonymes, c’est-à-dire sans connexion avec un véhicule précis, avec le conducteur ou autres utilisateurs autorisés à manœuvrer le véhicule.

Pour utiliser tous les services Škoda Connect, l’option Envoyer données de position doit être activée.
