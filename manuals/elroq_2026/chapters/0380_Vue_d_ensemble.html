Vue d’ensemble

Mode Eco

Le mode ECO est adapté à une conduite économe. La puissance du moteur est limitée, la vitesse maximale est réduite à 130 km/h. Le Climatronic passe en mode économique.

Mode Comfort

Le mode confort est adapté pour les déplacements sur les routes avec une surface en mauvaise état et pour les longs trajets.

Mode Normal

Le mode Normal est adapté à une conduite normale.

mode Sport

Le mode Sport est adapté à une conduite sportive.

mode Traction

Uniquement pour véhicules équipé d'une transmission intégrale.

Le mode Traction est adapté pour un trajet sur des routes lisses. La puissance est répartie de manière optimale entre les essieux avant et arrière.

Mode Individual

Le mode Individual permet de régler de manière personnalisée certains systèmes du véhicule.
