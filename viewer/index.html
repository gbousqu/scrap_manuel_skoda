ŠKODA Manuels locaux

Télécharger un manuel

Comme sur le site Škoda : saisissez votre code VIN ou sélectionnez un modèle, puis ouvrez le manuel. Le téléchargement ne démarre que lorsque le manuel s'affiche (…/show/…).

1

Ouvrir le portail Škoda

Un navigateur Chromium dédié s'ouvre sur le portail officiel des manuels.

2

Saisissez le code VIN

Entrez votre VIN à 17 caractères, choisissez la langue, cliquez sur Afficher.

ou

Sélectionnez un modèle

Parcourez la grille des modèles actuels (Enyaq, Elroq, Octavia…) et ouvrez le manuel.

3

Lancez le téléchargement

Quand l'URL contient digital-manual.skoda-auto.com/…/show/…, un bouton apparaît dans le navigateur.

Une fenêtre Chromium doit s'ouvrir sur votre bureau (pas dans cette page).

Le navigateur ne s'ouvre pas ?
  1. Cliquez Arrêter / réinitialiser
  2. Redémarrez WAMP (Restart All Services)
  3. Double-cliquez launch_scrape_task.bat dans le dossier du projet
  4. Ou en terminal : python scrape_manual_skoda.py
